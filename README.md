Project Title & Description: 
PDP Submission
This project contains the SQL script and R script for completing the PDP data submission process. 

Usage:
The SQL files are designed based on SIS database tables and used in Microsoft SSMS. The .R files are to be used in R studio. The process for creating the PDP cohort file includes running the SQL_for_Cohort_file code in SSMS, 
running the result file in R studio, then finializing the file with any PII that needs added and header/trailer rows. The same process is done for the course file.
